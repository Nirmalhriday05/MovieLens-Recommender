Models Implemented

1. Popularity-Based Recommender

Uses weighted rating formula (IMDB-style) combining average rating and vote count. Best for new users facing the cold-start problem.

2. User-Based Collaborative Filtering

Finds similar users using cosine similarity and recommends movies they enjoyed. Matrix size: 4,297 x 4,297 users.

3. Item-Based Collaborative Filtering

Computes movie-movie similarity based on rating patterns. More stable than user-based approach. Matrix size: 2,514 x 2,514 movies.

4. Matrix Factorization (SVD)

Decomposes user-item matrix into latent factors (50 dimensions) to discover hidden patterns in viewing preferences.

5. Content-Based Filtering

Recommends movies based on genre similarity using one-hot encoding and cosine similarity across 18 genres.

6. Hybrid Recommender

Combines popularity (30%), content-based (40%), and collaborative filtering (30%) for robust recommendations.
